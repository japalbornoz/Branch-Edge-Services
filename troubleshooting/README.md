# Troubleshooting Notes

This folder documents troubleshooting cases encountered while building and extending Project 04.

## Included Cases
- DHCP lease issues observed in Packet Tracer endpoint behavior during early Version 1 testing
- DNS reachability versus DNS application traffic behavior under ACL restrictions
- Version 2 DHCP relay failure caused by switchport mode mismatch between SW2 and R2
- Missing route conditions affecting return reachability to the remote guest subnet

## Goal
The purpose of this folder is to document how issues were isolated, what their root cause was, and how they were corrected.
